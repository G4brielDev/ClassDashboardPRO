# 🧠 Prompt Mestre para Desenvolvimento com IA

O **Prompt Mestre** é uma documentação criada para orientar modelos de Inteligência Artificial durante o desenvolvimento de um projeto de software. Seu objetivo é fornecer todo o contexto necessário para que a IA compreenda o escopo da aplicação e produza um código consistente, organizado e pronto para produção.

Ao invés de gerar apenas trechos isolados de código, este documento define toda a estrutura do projeto, reduz ambiguidades e garante que cada decisão tomada pela IA siga um mesmo padrão arquitetural.

## ✨ O que este Prompt Mestre define

Este documento descreve detalhadamente:

- 🎯 Objetivos do projeto e problemas que a aplicação resolve.
- 👥 Público-alvo e forma de utilização do sistema.
- 🛠️ Tecnologias obrigatórias e tecnologias que não devem ser utilizadas.
- ⚙️ Funcionalidades completas da aplicação.
- 📋 Regras de negócio e comportamentos esperados.
- 🗂️ Organização da estrutura de pastas.
- 🏛️ Arquitetura do projeto e padrões de desenvolvimento.
- 🧹 Convenções de código e boas práticas.
- 🎨 Diretrizes de design, identidade visual e experiência do usuário (UX/UI).
- 🗄️ Modelagem do banco de dados.
- 🔐 Estratégias de autenticação e autorização.
- 🌐 Integração com APIs e serviços externos.
- 🛡️ Regras de segurança e proteção de dados.
- ✅ Validações de formulários e entradas do usuário.
- ⚡ Estratégias de performance e otimização.
- 📱 Responsividade para diferentes dispositivos.
- ♿ Diretrizes de acessibilidade (A11Y).
- 🔍 Configurações de SEO quando aplicável.
- 🧪 Estratégias de testes automatizados.
- 🚀 Processo completo de deploy.
- 📚 Estrutura da documentação do projeto.
- 📈 Planejamento para escalabilidade e futuras versões.
- 🚧 Limitações técnicas e restrições do sistema.
- 📦 Requisitos para entrega final do projeto.

## 🚀 Benefícios

Utilizar este Prompt Mestre proporciona diversas vantagens durante o desenvolvimento assistido por IA:

- Redução significativa de retrabalho.
- Código mais organizado e padronizado.
- Arquitetura consistente desde o início do projeto.
- Melhor qualidade das respostas geradas pela IA.
- Facilidade para manutenção e evolução do sistema.
- Menor necessidade de ajustes manuais.
- Desenvolvimento mais rápido e previsível.
- Projeto preparado para crescimento futuro.

## 🎯 Objetivo

Este documento serve como a principal referência para a IA durante todo o ciclo de desenvolvimento. Antes de gerar qualquer código, a IA deve analisar integralmente este Prompt Mestre para compreender o contexto do projeto e implementar cada funcionalidade de forma completa, seguindo todas as especificações aqui definidas.

O resultado esperado é um projeto profissional, organizado, escalável e pronto para utilização em ambiente de produção.
